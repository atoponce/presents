The Value of Free Software, Part 1
==================================

* Example, LLC using "Brand X" RDBMS.
* Fast releases, low bugs, quick support.
* Annual renewing license contract.

The Value of Free Software, Part 2
==================================

* "Brand X" RDBMS goes bankrupt
* No releases.
* No security fixes.
* No support.
* Time to migrate.
* Problem? Vendor lock-in.


The Value of Free Software, Part 3
==================================

* Example, LLC using MySQL RDBMS.
* Fast releases, low bugs, quick support.
* Source code available.
* 3rd party support available.
* Community support available.
* Enterprise subscription available.

3. Distribute your modified versions (need source code).
